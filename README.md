# MALFY
## Created by Alisdair Chauvin 
*03/24/26*

This is my original synthesizer that can utilize FM synthesis to create unique timbre and enable visualization
through an oscilloscope. I created MALFY using the JUCE Framework and many JUCE and YouTube and Audio Programmer 
tutorials.

*Usage*
I coded this plugin using C++, so there is an accommodating CMakeLists.txt file that builds this plugin. It is
currently setup to build an AU, VST3, and Standalone version. The standalone is the one I used to troubleshoot, the AU
can be used in Logic specifically, and the VST3 version can be used in any other DAW.

To use, setup a MIDI instrument and connect through JUCE settings. Mess with the sliders to make cool sounds! Feel free
to download and experiment yourself.
